# How to apply this patch (GitHub web upload)

1. Download and unzip this patch zip locally.
2. In your repo on GitHub, click **Add file → Upload files**.
3. Drag-and-drop the folders/files from this patch (e.g. `.github/`, `src/`, `.gitignore`, `README.md`), preserving folder paths.
   - GitHub will show the path changes; confirm they line up with your repository root.
4. Commit the changes.

Notes:
- This patch **does not change the save schema**.
- On some systems, folders/files starting with a dot (like `.github` and `.gitignore`) may be hidden in the file picker. If you don’t see them, enable "show hidden files".
